This project is very straight-forward and user friendly. It allows you to design and simulate circuit using different gates.
Following instruction help you navigate your way through the website. It is JavaScript-Django based tool and simulation engine which make solving problems easy. You can create circuits and convert them into expression and truth table same goes for other two. You can input Boolean expression or truth table and converter them into remaining two.

Get to Converter

The tool consist of mainly three pannels: Boolean expression Converter, Truth Table Converter and last but not least Circuit Converter.
hover mouse on top of "converter" option on top of page that shows when you open the website. It will give three options

  • Boolean Expression Converter:

    By clicking on Boolean expression option, it will take you to the page wher you can give input in boolean expression.
    and then press to button to either convert it into truth table or circuit.

    1. Truth Table Converter:

      By clicking on Boolean expression option, it will take you to the page wher you can give input in Truth table.
      and then press to button to either convert it into Boolean Expression or circuit.

    2. Logic Circuit Converter:

      By clicking on Boolean expression option, it will take you to the page where you can create a circuit using different gates.
      and then press to button to either convert it into Boolean Expression or Truth Table.

    3. Working with Boolean Expression

    4. In order to input boolean expression; user need to type correct boolean expression that has same number of opening brackets as closing brackets.
      then press button available, to generate correct truth table or circuit of the expression.

    • Working with Truth Tables

  • In order to input Truth Table; first user need to input number of variables according to that a table will be generated. which Users will fill who they wish to with 0's and 1's
    then press button available, to generate correct Boolean Expression or circuit of the Truth Table.

  • The left part of the abstracted circuit contains the inputs, and the right side the outputs.

  • Click on the circle of the input or output to another input/output to make a connection.

    Working with Logic Circuits

  • Different devices that will be used to generate circuit are;

    • -in: to input the circuit

    • -out: to get output of circuit

    • -AND: to apply AND operator to the input between 1 and 0.

    • -OR: to apply OR operator to the input between 1 and 0.

    • -NOT: to apply NOT operator to the input between 1 and 0.

    • -XOR: to apply XOR operator to the input between 1 and 0.

  • And different gate functionality can be applied

  • Circuits will be created using these devices, then available buttons will be pressed, to generate correct Boolean Expression or Truth Table of that circuit

  • If you want to delete some device then drag it back to the sidebar and it will disappear.
